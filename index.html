1 Introduction

This chapter provides a foundation for our study of differential equations. We will return to the themes in this chapter throughout the book. A differential equation is an equation that relates an unknown function and its derivatives. The goal is to find a function (or, more typically, a set of functions) that satisfies this equation.

Remark 1.

Typically in this class, the variable “x” will be used to denote the dependent variable and “t” will be the independent variable (so, x=x⁢(t) means that x is a function of the variable t). But we will also use y is the dependent variable and x or t as the independent variable. See the examples below.
